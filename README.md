# Tic-Tac-Toe Java Game

A simple Tic-Tac-Toe game built using Java.

## Features
- Two-player game
- Console-based gameplay
- Player turns (X and O)
- Simple game board

## Technologies Used
- Java
- IntelliJ IDEA

## What I Learned
- Java fundamentals
- Arrays
- Methods
- Loops and conditions
